# Description

เป็นพื้นที่เก็บความรู้ ความประทับใจ สิ่งที่ได้ output ต่างๆ ที่ได้จากการอ่านหนังสือเล่มต่างๆ คือ repository นี้เกิดจากการที่ เรามีกองดองจำนวนมาก และรู้สึกว่าถ้าตะบี้ตะบันอ่านเพื่อเครียร์กองดองอย่างเดียว มันไม่ได้มีประสิทธิถาพในการเสียเงินเพื่อซื้อหนังสือแบบแล้วเหมือนเอาความรู้ผ่านหัวในชั่วครู่ ผ่านเข้ามาพักในสมองแปปๆ แล้วก็ลืมเลือน

สิ่งที่ทำให้อยากเขียนในนี้คือ เราอ่านหนังสือแบบ Physical แต่อยากเอาสิ่งที่ได้มา มาเก็บผ่าน Digital และเป็นเหมือน New year goals สำหรับ 2026 ว่าอยากจะเครียร์กองดองออกจากตั้งหนังสือ เลยวางแผนว่าถ้าอ่านจบบทแล้วจะมาเขียนบันทึกทำเป็นสิ่งที่ได้ในบทนั้นเก็บไว้
